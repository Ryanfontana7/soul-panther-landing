# Soul Panther — Landing

Static landing page + essays for the Soul Panther music project. Hand-authored HTML/CSS (no build step), designed in Open Design.

- `index.html` — landing (hero, bio, music, film, gallery, words, connect)
- `words.html` — essay index
- essays: `the-line-in-the-sand.html`, `ancient-and-modern.html`, `expensive-through-emptiness.html`, `ma-negative-space.html`, `mythic-in-the-club.html`, `words-post.html`
- gallery images (downscaled for web)

## Deploy

Static site on Vercel — no build, output is the repo root. Pushes to `main` auto-deploy.
